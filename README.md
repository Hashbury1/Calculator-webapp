# A Simple Calculator Web application


This is a clean, responsive web-based calculator designed for performing basic arithmetic operations (addition, subtraction, multiplication, and division). It is implemented using standard web technologies (HTML, CSS, JavaScript) and focuses on fast, reliable, and accessible calculations.

Since this is a simple static application, no server or build process is required. You can run the calculator directly in any modern web browser.

Prerequisites
You only need a modern web browser (Chrome, Firefox, Edge, Safari, etc.).

Installation (Local Setup)
Clone the repository to your local machine:

Bash

git clone https://github.com/Hashbury1/simple-calculator.git
Navigate into the project directory:

Bash

cd simple-calculator
Open the file index.html in your web browser.

(Tip: You can usually right-click the file and select "Open with" > [Brave or Chrome Browser].)
